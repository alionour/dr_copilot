# Project Documentation

This folder contains documentation for the project, organized into subfolders:

- `architecture/`: High-level and detailed architecture documents.
- `onboarding/`: Guides and checklists for new contributors.



Other key documents:
- `architecture/APP_CONTEXT_SUMMARY.md`: Application context and summary.
- `architecture/BILLING_SUMMARY.md`: Billing and payment flow.
- `onboarding/APP_ICON_GENERATION.md`: App icon generation process.
- `onboarding/RELEASE_CHECKLIST.md`: Steps for releasing the app.
- `architecture/ROLES_PERMISSIONS_STRATEGY.md`: Roles and permissions strategy.

Feel free to add more folders or files as the project grows.
